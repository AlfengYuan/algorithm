◆ mqtt_version()

auto mqtt::connect_options_builder::mqtt_version ( int  ver) -> self&
inline

Sets the version of MQTT to be used on the connect.

This will also set other connect options to legal values dependent on the selected version.

Parameters
verThe MQTT protocol version to use for the connection:
  • MQTTVERSION_DEFAULT (0) = default: start with 3.1.1, and if that fails, fall back to 3.1
  • MQTTVERSION_3_1 (3) = only try version 3.1
  • MQTTVERSION_3_1_1 (4) = only try version 3.1.1
  • MQTTVERSION_5 (5) = only try version 5
Deprecated:
It is preferable to create the options builder for the desired version rather than using this function to change the version after some parameters have already been set. If you do use this function, call it before setting any other version-specific options.
See also
connect_options_builder::v5()
